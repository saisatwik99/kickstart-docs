Folder Structure

Let us understand the folder structure.

Folder Structure:
node_modules This folder contains all the packages we are using to run the application.
public This folder is created automatically by create-react-app.
server This folder contain the backend server. Which is written in Node.JS.
src This folder all the React components.
babel-plugin-macros.config.js This file configures the tailwind.config.js.
jsconfig.json This file helps to recognise the react components(src folder).
package-lock.json Automatically generated file to operate node modules.
package.json Package.json specify the version of a package that our app depends on.

Our application is written in React.js.
